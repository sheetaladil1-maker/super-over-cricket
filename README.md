# Super Over — Retro Cricket (HTML/JS)

**Super Over** is a retro-style 2-player browser cricket game. Two innings (6 balls each), random runs and OUT, target chase, confetti on 4/6, ball-by-ball commentary, Chart.js scoreboard, retro winner animation and restart.

## How to run
Open `index.html` in a modern browser (Chrome/Firefox/Edge). Click **Start Match** then **Hit (Play Ball)**.
